Bee  is an ORM framework.

Bee is Tea: (Timesaving/Tasteful, Easy, Automatic) style ORM framework.

Bee see: https://github.com/automvc/bee

-----------------------------

Bee 是一个 ORM框架,具有:

省时/优雅、简易、自动 (Tea:Timesaving/Tasteful, Easy, Automatic) 风格的ORM框架。

入门快速,使用容易,编码量少。

Bee 网址: https://github.com/automvc/bee

Bee在gitee 的网址: https://gitee.com/automvc/bee

--------------------

直接运行org.teasoft.honey.osql.example包下的例子,即可体验bee框架是如何使用的.
