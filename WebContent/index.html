Bee  is an ORM framework.

Bee is I Sea (Intuitive,Simple, Easy, Automatic) style ORM framework.

Bee see: https://github.com/automvc/bee

-----------------------------

Bee 是一个 ORM框架.
Honey 是Bee的实现.
Bee 是一个直观、简单、易用、自动(I Sea: Intuitive, Simple, Easy, Automatic) 风格的ORM框架.
Bee 网址:
https://github.com/automvc/bee

--------------------

直接运行org.honey.osql.example包下的例子,即可体验bee框架是如何使用的.
